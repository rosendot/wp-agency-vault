# WP Agency Vault

A personal knowledge base and reusable toolkit for building WordPress client sites. When starting a new client project, pull templates, guides, and snippets from here instead of starting from scratch.

## What's Inside

| Folder | Purpose |
|--------|---------|
| `guides/` | Step-by-step reference docs — Bedrock setup, LocalWP config, deployment, CPTs |
| `boilerplate/` | Starter files to copy into new projects — theme skeleton, `.env` template, Claude config |
| `snippets/` | Copy-paste code patterns organized by language (PHP, CSS, JS) |
| `plugins/` | Notes on plugins you've vetted and when to use them |

## How to Use

1. **Starting a new client project?** Copy files from `boilerplate/` into your new repo
2. **Need to set something up?** Check `guides/` for step-by-step instructions
3. **Building a feature?** Grab a snippet from `snippets/` and adapt it
4. **Choosing a plugin?** Check `plugins/recommended.md` for vetted options

## Contributing to the Vault

Every time you learn something or solve a problem on a client project, save the reusable version here:

- Solved a tricky layout? Add it to `snippets/css/`
- Figured out a WP query pattern? Add it to `snippets/php/`
- Set up hosting for a client? Write it up in `guides/`
- Found a great plugin? Add it to `plugins/recommended.md`

Over time this becomes your agency's competitive advantage — you ship faster because you're not figuring things out from zero each time.
