# Performance Improve

Reads the `perf-report.md` generated by the Performance Analyzer agent and applies the fixes.

## Usage
`/perf-improve <template-slug>`

## Prerequisites
The Performance Analyzer agent must have been run first. This skill reads `perf-report.md` from the template directory. If the report does not exist, tell the user:

> Run the Performance Analyzer agent on this template first. It will generate the `perf-report.md` that this skill needs.

## Steps

1. Read `templates/$ARGUMENTS/perf-report.md` (or the path the user specifies).
2. Parse all **Critical** and **Warning** issues from the report. Each issue has:
   - File path
   - Line number(s)
   - Affected metric (LCP/CLS/INP)
   - Problem description
   - Fix instruction
3. For each issue, starting with Critical then Warnings:
   - Read the referenced file
   - Apply the fix described in the report
   - Verify the fix doesn't break functionality
4. After all Critical and Warning fixes are applied, list the **Suggestions** and ask the user which ones they want applied.
5. After all chosen fixes are applied, delete `perf-report.md` — it is now stale.
6. Summarize what was changed:
   - Files modified
   - Issues fixed (by severity and metric)
   - Expected Core Web Vitals impact
   - Suggestions skipped or applied

## Fix Patterns

### Move scripts to footer
```php
// Before
wp_enqueue_script( 'template-main', get_template_directory_uri() . '/js/main.js', array(), '1.0' );
// After
wp_enqueue_script( 'template-main', get_template_directory_uri() . '/js/main.js', array(), '1.0', true );
```

### Add defer to non-critical scripts
```php
add_filter( 'script_loader_tag', function( $tag, $handle ) {
	$defer_handles = array( 'template-carousel', 'template-lightbox' );
	if ( in_array( $handle, $defer_handles, true ) ) {
		return str_replace( ' src', ' defer src', $tag );
	}
	return $tag;
}, 10, 2 );
```

### Replace @import with enqueue dependencies
```php
// Before (in style.css): @import url('carousel.css');
// After (in functions.php):
wp_enqueue_style( 'template-carousel', get_template_directory_uri() . '/carousel.css', array(), '1.0' );
wp_enqueue_style( 'template-style', get_template_directory_uri() . '/style.css', array( 'template-carousel' ), '1.0' );
```

### Add image dimensions
```html
<!-- Before -->
<img src="photo.jpg" alt="Description">
<!-- After -->
<img src="photo.jpg" alt="Description" width="800" height="600">
```

### Fix lazy loading on hero images
```html
<!-- Before — hurts LCP -->
<img src="hero.jpg" alt="Hero" loading="lazy">
<!-- After -->
<img src="hero.jpg" alt="Hero" fetchpriority="high">
```

### Add lazy loading to below-fold images
```html
<!-- Before -->
<img src="gallery-1.jpg" alt="Gallery photo">
<!-- After -->
<img src="gallery-1.jpg" alt="Gallery photo" loading="lazy">
```

### Extract large inline styles
Move inline `<style>` blocks larger than ~500 bytes into the template's CSS file and enqueue properly.

### Font loading optimization
```html
<!-- Add to header.php before wp_head() -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
```
And ensure `font-display: swap` in @font-face declarations.

### Conditional asset loading
```php
// Before — loads on every page
wp_enqueue_script( 'template-carousel', ... );
// After — only loads where needed
if ( is_front_page() || is_page_template( 'page-gallery.php' ) ) {
	wp_enqueue_script( 'template-carousel', ... );
}
```

## Rules
- Only fix issues listed in `perf-report.md` — do not go looking for additional issues
- Apply Critical and Warning fixes without asking. Ask before applying Suggestions.
- Never recommend caching plugins or server-side changes — Cloudways handles that
- Never recommend image compression plugins — focus on template code only
- All PHP must remain properly escaped and follow WordPress coding standards
- Delete `perf-report.md` when done — a stale report is worse than no report
- If a fix could change visual appearance or break functionality, flag it to the user instead of guessing
